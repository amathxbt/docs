## Multi-AMM Liquidity

On graduation, ACTFUN seeds liquidity on **three AMMs simultaneously**:

| AMM | Type | Pool per Token |
| --- | --- | --- |
| **UNITFLOW V3** | Uniswap V3 fork (concentrated liquidity) | Full-range position |
| **Uniswap V2** | Classic constant-product (xГ—y=k) | New pair per token |
| **Curve Finance (StableSwap)** | Low-slippage pool for correlated assets | **New pool per token** via StableSwapFactory |

This means every graduated token has its own dedicated Curve Finance pool, providing deep liquidity for low-slippage swaps.

## Architecture

### How Graduation Works

1. **Create** вЂ” Anyone deploys a token via LaunchpadFactory (free on testnet)
2. **Mine** вЂ” Community mines 95% of supply by writing funny posts on-chain
3. **Graduate** вЂ” At 95% mined, the contract auto-graduates:
   - Mints 5% LP reserve tokens to itself
   - Seeds liquidity on **UNITFLOW V3 + Uniswap V2 + Curve Finance**
   - Each token gets its own Curve Finance pool via StableSwapFactory
   - LP positions are locked in the contract forever
4. **Trade** вЂ” Community buys/sells via the built-in DEX or chooses any of the three AMMs
